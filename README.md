# Fair9

**The Open Source, Ultra-Low Latency Voice Assistant.**

Fair9 is a next-generation voice interface built with **Flutter** (UI) and **Rust** (Engine) to deliver instantaneous transcription and command execution.

## Features
- **Zero-Latency Transcription**: Powered by `cpal` and `whisper-rs` (Rust).
- **Glassmorphic HUD**: A beautiful, frameless UI that floats on your desktop.
- **Privacy First**: All processing runs locally. No cloud API keys required.
- **Smart Formatting**: Context-aware punctuation and capitalization.
